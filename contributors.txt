Simple Javascript Web app that:
  -Uses GoogleMaps JS API and geometry library
  -Use teh air-port-codes.com api to grab all the worlds airport codes
  -Uses custom filter function to prevent data sources from outside US airports
  -plots path using GoogleMaps PolyLine
  -converts length to Nautical Miles
